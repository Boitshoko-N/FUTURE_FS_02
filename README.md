# 📊 CRM Dashboard App

A simple Customer Relationship Management (CRM) web application built with **React**, **Node.js (Express)**, and **MongoDB**.  
The app allows users to create, view, and update leads in a central dashboard.

---

## 🚀 Features

- ➕ Add new leads
- 📋 View all leads in a list
- ✏️ Update lead information
- 🗄️ MongoDB database integration
- 🔗 REST API built with Express
- ⚡ React frontend with Axios API calls
- 🌐 CORS enabled for frontend-backend communication

---

## 🛠️ Tech Stack

### Frontend
- React
- Axios
- JavaScript (ES6)

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
